# 佐罗上网管控 (ZNetControl)

基于MAC/IP地址的网络访问时间管控系统，支持按时间段、星期和日期范围精确控制设备上网。

## 功能特点
- 支持MAC地址和IP地址双模式管控
- 按时间段控制（开始时间/结束时间）
- 按星期控制（可设置工作日、周末或特定星期）
- 实时监控和自动规则切换
- 完整的Web管理界面
- 详细的系统日志
