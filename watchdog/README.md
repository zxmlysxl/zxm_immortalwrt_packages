### Plugin features: Solve the security login control problem of OPENWRT, adapt to OpenWR24.10, automatically adapt to Iptablet FW3 and NFT FW4. Can monitor WEB pages and SSH login status. Automatically blacklisted after multiple failures.


### Method for downloading source code:

 ```Brach
    # downloading
    git clone https://github.com/sirpdboy/luci-app-watchdog package/watchdog
    make menuconfig
	
 ``` 
### Configuration Menu
 ```Brach
    make menuconfig
	# find LuCI -> Applications, select luci-app-watchdog, save and exit
 ``` 
### compile

 ```Brach 
    # compile
    make package/watchdog/luci-app-watchdog/compile V=s
 ```
